## Setup

1. Clone the project.
```sh
https://github.com/azdanov/translations.git
cd translations
cp .env.example .env
```

2. NPM Scripts:
```sh
# Project setup
yarn

# Compiles and hot-reloads for development
yarn start

# Compiles and minifies for production
yarn build

# Run your tests
yarn test
```
